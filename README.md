Passport Management System

📖 Project Overview

The Passport Management System is a Java web application developed to simplify the passport application process. It provides separate modules for Users, Officers, and Administrators, allowing passport applications to be submitted, reviewed, approved, and managed efficiently.

This project was built as a learning project to gain practical experience with Java web development.

---

🚀 Technologies Used

- Java
- JSP (JavaServer Pages)
- Servlets
- JDBC
- SQLite
- HTML
- CSS
- JavaScript
- Apache Tomcat
- NetBeans IDE

---

👥 Modules

👤 User Module

- User Registration
- User Login
- Apply for Passport (Multi-step Application)
- Preview Application
- Submit Application
- View Application Status

👮 Officer Module

- Officer Login
- View Passport Applications
- Approve Applications
- Reject Applications

🛡️ Admin Module

- Admin Login
- View All Passport Applications
- View Officers

---

▶️ How to Run

1. Import the project into NetBeans.
2. Add the SQLite JDBC library.
3. Place the SQLite database ("pmsdb.db") in the required location.
4. Configure Apache Tomcat.
5. Run the project.
